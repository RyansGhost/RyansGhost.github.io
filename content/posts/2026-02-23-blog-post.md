# A Beginner's Guide to C++

Welcome to the world of C++! This guide is designed for beginners to get started with C++ programming.

## 1. Hello World

The simplest C++ program:

```cpp
#include <iostream>
int main() {
    std::cout << "Hello, World!";
    return 0;
}
```
